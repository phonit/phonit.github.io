实时进程类别  class

SCHED_FIFO :first in first out        FF   实时优先级

SCHED_RR:  round robin              RR   实时优先级

SCHED_OTHER:  调度用户空间的100-139的    TS

SCHED_OTHER中有个动态优先级，长时间未被调度的进程，临时性的调高优先级运行。

 

手动调整优先级

nice 100-139  (-20-19)

nice -n -20 ping -f 127.0.0.1   新建的

renice -n -10 pid                  修改运行中的

chrt -p [prio] pid

值越小，进程”优先级”越高，而值越大“优先级”越低。

 

 

chrt  1-99   实时优先级

chrt -f -p [prio]  pid   调整 FIFO类别

chrt -r -p [prio]  pid   调整RR类别    实时优先级

chrt -f -p [prio]  cmd  实时启动一个优先级
