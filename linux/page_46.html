创建用户模板

/etc/default/useradd  创建用户时，默认目录、组选项

/etc/login.defs            创建用户时，默认用户权限

/etc/skel/*                 登录用户的家默认文件

 

批量修改用户密码

newusers < user.txt                            批量创建用户   passwd文件的格式

cat list.txt |chpasswd                             批量修改密码     username:passwd

echo 密码 | passwd username --stdin    可视化修改密码

chage -d 0 username                           下次登录强制修改密码

usermod -U  解锁 、passwd -l  解锁

 

pkill -kill -t tty参数  踢出用户

 

单次执行root 命令

su - root -c 'cat /etc/shadow'
