下载       git fetch更安全一些。因为在merge前，我们可以查看更新情况，然后再决定是否合并结束

git fetch     从远程获取最新版本到本地，不会自动merge

git pull：相当于是从远程获取最新版本并merge到本地

 

举例说明：将远程origin仓库的master分支合并到本地的yy分支。
git fetch origin master
git checkout yy
git merge FETCH_HEAD
